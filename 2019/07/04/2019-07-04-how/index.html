使用Hexo建置網站

Posted on 2019-07-04 | In Hexo |

參考資料

  • youtube：使用 Hexo 建構個人化部落格
  • 共筆文件：講義

軟體工具準備

  • 安裝 Node.js
  • 安裝 git
  • 使用工具：visual studio code
  • Hexo：文件

Hexo流程

開啟CMD or VS code 終端機

cd <folder>
// 切換到要使用的資料夾

安裝 hexo cli

npm install -g hexo-cli
//可以讓 hexo使用指令的工具

建立雛形資料夾

hexo init <folder>  //無指定預設建立在原資料夾中
cd <floder> //無指定，此步驟省略
npm install //完成安裝

啟動本地預覽

hexo s  
// s=sever , localhost:4000

建立靜態網站

hexo g
// g=generate ,生成 public 資料夾

佈署至外部網站

申請github帳號：github

連結至git

// create a new repository
git init
git add . 
git commit -m "自行輸入說明"

上載原始碼

// push an existing repository
git remote add origin https://github.com/xxxx/xxx.github.io.git //儲存路徑
git push -u origin master

發佈網站

// 上載 public 資料夾    
hexo d
